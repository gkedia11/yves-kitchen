# Yve's Kitchen Landing Page

A high-end, ultra-modern landing page for Yve's Kitchen, featuring a "Dark Mode Luxury" aesthetic.

## Features
- **Immersive Design**: Dark mode with gold accents and high-contrast typography.
- **Parallax Scrolling**: Smooth visual effects on scroll.
- **Masonry Gallery**: Responsive grid for food photography.
- **Menu Sections**: Categorized menu items with image reveals.
- **Interactive Elements**: Custom cursor and hover effects.
- **WhatsApp Integration**: Direct ordering via WhatsApp API.

## technologies Used
- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (Intersection Observer, Parallax logic)
- Google Fonts (Playfair Display, Plus Jakarta Sans)

## How to Run
Simply open `index.html` in any modern web browser.

## Note on Images
The project currently uses high-quality placeholder images from Unsplash matching the descriptions (Suya, Meat Pie, Grilled Fish, etc.) because the custom image generation service was temporarily unavailable. You can replace these images in `index.html` and `style.css` (for the hero background) with your own professional photography as needed.

## Editing Content
- **Menu Items**: Edit the HTML in the `#menu` section.
- **Contact Info**: Update phone numbers and links in the `<header>` and `<footer>`.
- **Address**: Update the address in the `<footer>`.
